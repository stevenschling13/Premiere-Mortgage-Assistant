# Contributing

Thank you for contributing to Premiere Mortgage Assistant.

- Branching: Use feature branches named feat/<short-desc> or fix/<short-desc>.
- Pull requests: Target `main`. Provide description, testing steps, and reference issue(s).
- Code style: Run `npm run lint` and `npm run format` before committing.
- Commits: Use Conventional Commits (feat, fix, chore, docs, refactor, test).
- Tests: Add unit tests for new logic. CI enforces tests for PRs.
- Security: Do not commit secrets or PII. Store secrets in GitHub Actions Secrets or an approved vault.
